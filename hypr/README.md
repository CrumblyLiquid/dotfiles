# Hyprland

[Hyprland](https://hypr.land/)
is a Wayland based dynamic tiling compositor
with animations!

## Packages

- `hyprland`
- `hypridle` (idle lock)
- `hyprlock` (lockscreen)
- `hyprpaper` (wallpaper)
- UWSM support
    - `uwsm`
    - `libnewt`
- `grimblast` (screenshots)
- `playerctl` (music control)

## Configuration

Main configuration entrypoint for Hyprland is
`hyprland.conf` usually located under `~/.config/hypr/`.
Other Hypr components use their similarily named
files (`hypridle.conf`, `hyprlock.conf`, `hyprpaper.conf`).
